# JS Afro General Trading FZC LLC - Corporate Website

A modern, professional, and trust-focused corporate website for JS Afro General Trading FZC LLC, a UAE-based multi-service trading and solutions company.

## Features

- **Responsive Design**: Fully optimized for desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean layout with professional spacing and typography
- **SEO-Friendly**: Semantic HTML structure with proper meta tags
- **Fast Loading**: Optimized images and minimal dependencies
- **Interactive Elements**: Smooth scrolling, animations, and hover effects
- **Contact Form**: Integrated contact form for customer inquiries
- **Google Maps**: Embedded map showing Sharjah location

## Sections

1. **Home** - Hero section with compelling headline and CTAs
2. **About Us** - Company overview and core values
3. **Services** - 7 comprehensive service offerings:
   - Mail Order Houses
   - E-Commerce Platform
   - Online Retail & IT Solutions
   - Social Media E-Commerce
   - Freight Forwarding Services
   - Procurement Consulting
   - Medical Treatment & Overseas Facilitation
4. **Corporate Responsibility** - Environmental and ethical commitments
5. **Contact** - Contact information, form, and map

## Setup

1. Clone or download the repository
2. Open `index.html` in a web browser
3. No build process or dependencies required

## Customization

### Updating Content
- Edit `index.html` to modify text content
- Update contact information in the Contact section
- Replace placeholder images with company-specific photos

### Styling
- Modify CSS variables in `styles.css` for color scheme changes
- Adjust spacing, typography, and layout in the CSS file

### Functionality
- Customize form handling in `script.js`
- Add analytics tracking or additional interactive features
